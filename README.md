# Brazy Test Task

Приложение для управления и просмотра транзакций с фильтрацией и детальной информацией.

## Быстрый старт

```bash
npm install
npm run dev
```

## Технологический стек

- **React**
- **TypeScript** 
- **Vite** 
- **Context API**
- **Ant Design**
- **Day.js**
- **SCSS Modules** - (два файла, можно было и без них обойтись)
- **FSD** 

### Технические особенности
- 10% вероятность ошибки запроса для тестирования обработки ошибок
- В (`src/features/transactions/context`) реализована оснавная логика филтрации и загрузки транзакций
- Сортировка реализовано при помощи Ant-desing прямо в таблицах
  
### Не реализовано (опционально)
- Unit/Integration тесты
- Timeline в детальной информации

## Структура проекта

```
src/
├── api/           # API и моковые данные
├── app/           # Корневой компонент приложения
├── features/      # Бизнес-логика (транзакции)
├── pages/         # Страницы приложения
├── shared/        # Переиспользуемые/вынесеные компоненты и утилиты
└── widgets/       # Сложные компоненты (таблицы, фильтры, модалки)
```

## Технические решения

**Ant Design** - выбран за готовый набор компонентов, соответствующих UX требованиям проекта (Table, Modal, DatePicker, Filters и др.)

**Context API** - достаточно для управления небольшим количеством состояний, не требует дополнительных зависимостей
